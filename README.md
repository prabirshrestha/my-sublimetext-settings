## Installing on Mac

```bash
cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages/
rm -rf User/
git clone git@github.com:prabirshrestha/my-sublimetext-settings.git User
```

## Installing on Windows

```bat
cd %APPDATA%\Sublime Text 2\Packages
del /s User
rmdir User
git clone git@github.com:prabirshrestha/my-sublimetext-settings.git User
```
